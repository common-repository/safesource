== Description ==

Simple method to protect sourcecode, content and images from direct theft

Prevent Content and Source theft. 
Removes 'Right Mouse Click' Option. 
Removes 'Image Save' Options. 
Removes ability to highlight text anywhere on page except forms.
Removes ability to 'hold to copy' on Mobile Devices.

This plugin is really easy to use, just activate and go.

== Installation ==

Install, activate and leave to work its magic.
